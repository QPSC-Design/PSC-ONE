# なぜPSC-ONEを作るのか

AIは急速に私たちの生活へ入り込んでいます。

音声認識、画像認識、翻訳、文章生成。

今やAIは特別な研究設備ではなく、誰もが利用する時代になりました。

しかし、そのAIがどこで動いているのかを意識したことはあるでしょうか。

多くの場合、私たちが利用するAIは巨大なクラウドサーバの中で動いています。

- どのようなモデルが使われているのか
- どのようなデータが送信されているのか
- どのような処理が行われているのか

ユーザーからは見えません。

ブラックボックスです。

もちろん便利です。

しかし、本当にそれだけで良いのでしょうか。

## 自分だけのAIを持つ

私は、

**「自分のポケットの中に、自分だけのAIを持ちたい」**

と考えました。

- インターネット接続がなくても動くAI
- 自分で作ったAI
- 自分で学習させたAI
- 自分で改造できるAI
- その中身をすべて理解できるAI

スマートフォンで十分だ、という考えもあるでしょう。

確かにスマートフォンは高性能です。

しかしスマートフォンは完成品です。

CPUもAIエンジンもOSもブラックボックスです。

私たちは利用者であって、設計者ではありません。

## PSC-ONEの目的

PSC-ONEは違います。

CPUコア。

AIアクセラレータ。

SDカードインターフェース。

メモリコントローラ。

LCD制御回路。

OSやソフトウェアまで。

そのすべてをオープンソースで公開し、自分自身で設計・理解・改造できることを目指しています。

## AIを理解するためのプラットフォーム

AIを使う時代から、

**AIを理解する時代へ。**

AIを個人が所有する時代へ。

AIを自分で作る時代へ。

PSC-ONEは、そのための小さな一歩です。
