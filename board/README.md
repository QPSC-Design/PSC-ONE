# PSC-ONE Board

## PSC-ONE

The first-generation PSC hardware platform designed for CPU, operating system, memory subsystem, peripheral, and AI accelerator development and validation.

PSC-ONE serves as the primary evaluation board for the PSC ecosystem and provides a complete hardware/software co-design environment for FPGA-based system development.
  
The current PSC-ONE prototype hardware.  
  
<img src="/docs/images/PSC-ONE_board.jpg" width="600">

## PSC-ONE Neo (Planned)

The next-generation PSC platform featuring improved performance, expanded memory capacity, enhanced AI acceleration capabilities, and additional peripheral interfaces.

Unlike the original prototype platform, PSC-ONE Neo is designed with the FPGA mounted directly on the PCB, enabling lower manufacturing costs and supporting small-scale production.

PSC-ONE Neo is intended as the evolution of the original PSC-ONE architecture and serves as the main development platform for future PSC technologies.

## PSC-Robot (Planned)

A robotics-focused PSC platform designed for autonomous systems, voice recognition, sensor fusion, motor control, and edge AI applications.

PSC-Robot demonstrates the real-world capabilities of PSC hardware through mobile robotics and intelligent embedded system applications.
